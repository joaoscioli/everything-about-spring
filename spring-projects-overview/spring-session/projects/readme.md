# Spring Projects Showcase

This directory contains practical Spring projects designed to demonstrate different parts of the Spring ecosystem.

The goal is to evolve from foundational concepts to advanced distributed systems.

Each project focuses on a specific domain of the Spring ecosystem.

---

# Project List

## 01 - Spring Core Demo

Demonstrates:

- Inversion of Control (IoC)
- Dependency Injection (DI)
- Bean lifecycle
- Bean scopes

Focus: Understanding the Spring container fundamentals.

---

## 02 - REST API with JPA

Demonstrates:

- RESTful API development
- Layered architecture (Controller, Service, Repository)
- Spring Data JPA
- Validation and exception handling

Focus: Building production-ready CRUD APIs.

---

## 03 - Spring Security with JWT

Demonstrates:

- Authentication and authorization
- JWT-based stateless security
- Custom security filters
- Role-based access control

Focus: Securing APIs.

---

## 04 - Microservices with Spring Cloud

Demonstrates:

- Service discovery
- API Gateway
- Distributed configuration
- Circuit breakers
- Inter-service communication

Focus: Cloud-native microservices architecture.

---

## 05 - Reactive API with WebFlux

Demonstrates:

- Reactive programming
- Mono and Flux
- Non-blocking APIs
- Backpressure

Focus: High-performance reactive systems.

---

## 06 - Batch Processing

Demonstrates:

- Scheduled jobs
- Large data processing
- Retry mechanisms
- Fault tolerance

Focus: Enterprise batch workloads.

---

## 07 - Event-Driven Architecture with Kafka

Demonstrates:

- Producer/Consumer model
- Event-driven communication
- Asynchronous messaging

Focus: Distributed event-based systems.

---

## 08 - Hexagonal Architecture

Demonstrates:

- Clean architecture principles
- Ports and adapters
- Domain-driven design approach
- Testable core domain

Focus: Maintainable and scalable architecture.

---

# How to Use These Projects

Each project contains:

- Independent README
- Setup instructions
- Dependencies
- Architecture explanation
- Example requests

You can explore projects individually or follow them sequentially as a learning path.

---

# Learning Path Recommendation

1. Start with Spring Core
2. Move to REST API
3. Add Security
4. Explore Reactive
5. Study Microservices
6. Explore Event-Driven
7. Finish with Hexagonal Architecture
