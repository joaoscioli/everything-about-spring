# Spring Boot REST API

This project is a **foundational Spring Boot REST API** created as part of the  
**Everything About Spring Framework** series by *The Coding Cabin*.

Its purpose is to demonstrate how a real Spring Boot application starts:
with clear decisions, minimal dependencies, and a clean architectural baseline.
